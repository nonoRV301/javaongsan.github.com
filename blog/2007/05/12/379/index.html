this post is from my o2, so let me warn you in advance, it won’t be pretty.
if you are struggling to read 10-50or more rss feeds everyday, i got a
quick tip for u.

first of all, you need read up on post on hacking firefox for a better rss
reader. most rss readers are formated like emails, thus effort needed to
quick on each feed, each header. my hack allows you to see a list of header
with a extract of the post, so at one glance yoi see how many new post and
scan through them quickly.

next you need to arrange your feeds in 3 category, top which is author who
post daily and more than once, mid are those who post once a day. low, once
every few day
create a folder for each of them. everyday, You go through the folders from
low to high. use the open all tabs from firefox, it will open all the feed
in that folder. glance through all the header, if header interest you,
read the short description, click on those that you want to read, bookmark
them. once you go through all the feeds, you open all the articles you have
bookmark in tabs and go through tHem.

ok now, this tips is for firefox user only, for ie user you be able to find
the download button on the top of my page. the best part its works better
than ie and its free
