Looking for New Web Host

Oct 11th, 2007

![]Web hosting is important not only to the people who are using the website but the webmaster as well. I am currently fustrated by the issue with my web host. There was a problem connecting to my host and I have found this problem have always been there. Now that I have hosted my own blog, it is more obvious. I have to refresh my browser every time I make a connection, it would say error web server not available. I am using Godday.com Linux account, I can tell you they are not good enough.

I have look at bluehost. Bluehost was recommended by wordpress but seems to have read a lot of negative about it. The worst things is that theres only one price for there hosting. I have also look at bluefur, but I have not heard from them after signing up as an affiliate. Web hosting preferences have always been divided among webmasters. its mixed of good and bad in every company. I would actually prefer a local company to host my account, that way the access speed is higher. Thats nothing more frustrating than not able to publishing your work or losing it because of something like webhosting.

It be good to find a web host sponsor but right now I would like to know if theres a better company out there.

Update: I just received an email form bluefur.

Thank you for signing up for the bluefur.com Affiliate Program. We have reviewed your application and at this we do not feel there is a good match as partners. If you have any questions about why please direct them to affiliate@bluefur.com.

Sincerely,

bluefur.com Staff
