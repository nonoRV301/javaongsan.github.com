Freelancing 2 Years In

Jan 15th, 2014

This month was my second year as a freelance programmer. This are somethings that have worked well.

Recurring income

Always ask for more work from existing client. Charge a monthly for maintenances if you can.

Tools

Macbook air, Dell 23 inch have been the best investment ever. I hope to get a new MBA with more ram than the current 4g and a 27i nch Monitor thats now cost less than $500. A mac pro would be ideal but a spec up one is really costly.

Skills

C#, PHP and javascript are the most common skill I am using. I write a lot python scripts to process things.

Networking

I attended a lot more events and hackathons. I make some really good connections.

learning

I learn a lot of marketing techniques since.
