Ironman China and Ironman 70.3 China on the Same Day

Nov 26th, 2007

Ironman China & Ironman 70.3 China will both take place on April 20, 2008 and will be the first time that a full-distance Ironman and Ironman 70.3 race will be staged at the same venue on the same day. It going to be an awesome event. The event will be at a City of Haikou, the beautiful Hainan Island, known as Asia’s version of Hawaii.

The Ironman China website (www.IronmanChina.com) reports: “Our venue is world-class in all respects and among the best in the Ironman world”, said Murphy Reinschreiber, Ironman China’s Event Director. “Haikou is a city of 1.7 million people (it would be the 4th largest city in the US) with all the action and opportunities of a major city. Our host hotel, Sheraton Haikou Resort, compares with similar 5-star resort hotels in Hawaii, Phuket, Noosa and other resort destinations.”

Our bike course is primarily rural, touching the City at the beginning and end of each loop. We expect to use very few cones to separate bikes from cars because bikes will be assigned to entire sides of divided/closed access roads (2+ lanes). There is a section on narrow roads that climbs through two farming villages which will be packed with amazed on-lookers.

The run course includes Haikou’s version of Alii drive (Bin Hai Avenue), Central Park (Evergreen Park), and the Golden Gate Bridge (New Century Bridge) and then cuts right through the heart of the Old City. It is flat except for the Bridge, palm-shaded in stretches, and interesting to run.”

Ironman 70.3 China Ironman China
Ironman 70.3 China will start two hours after Ironman China. The course will be one-loop of Ironman China’s two-loop swim, bike and run course. Ironman 70.3 China finish line will be adjacent to Ironman China’s finish line. Both races will share the same post-race facilities and amenities such as massage. The 70.3 awards will be given in front of the Ironman China during the evening.

Ironman 70.3 China offers 25 qualifying slots for the Ford Ironman 70.3 World Championship in Clearwater, Florida, USA in November 2008.

Xampplite

Nov 17th, 2007

Xampplite is a cool tool. If you need to work on php and mysql and move your code around work stations,
xampplite is really a good software to use. Just Download it
and unzip it into your thumb drive at the highest level,
it must be on the highest level like E: or W:. It will build E:XAMPPlite or W:XAMPPlite or something like this.
Start xampp by using the control center, xampp-control.exe, start Aapche and mysql using the user friendly interface.
Start your browser and type http://127.0.0.1 or http://localhost in the location bar. You should see the pre-made
start page with certain examples and test screens. Stick your usb dive into any computer, start control and run your
php under Drive:XAMPPlitehtdocstest.php => http://localhost/test.php. You can setup wordpress too.
Just unzip the installation file under wordpress under htdocs, create database with phpmyadmin which can
be found on the default page http://localhost look under the left side bar.
I have been using Xampp to be build my own fast install version of wordpress. I coded some default values and removed annoying stuff like
default entries for blogroll. I hack an rss scrapper plugin. The goal is to be able to set up subdomain blog scrappers in a few seconds.
The whole idea of scrapper is to find backlinks through trackbacks. If you add a like to another post you get a link back just like a comment,
except trackbacks are 2 way links. If you do the math, you scrape 10 post = 10 links, if you scrape 10 post from 10 blogs you get 100 links, get the point?
If you throw in some keywords and do an rss search you get few hundred rss links, each have between 10-20 post, you get few hundred links instantly from each blog.
You can see a sample of my work at cappadon.com. This a music aggreator site. The plan was to host mtv videos but I couldn’t
have time to look for a cms that will do it so for now its just a blog. This a used domain that I bought so I got some backlinks and all the traffic are from organic search engine.
Its a PR2 as well. Next, I will be working on forums like postnuke and cms like drupal. I will be done
with my day job next week, after that its back to full time online work.

Selecting a VPS

Oct 28th, 2007

I am shopping for a VPS. Anyone has any recommendations? A VPS Virtual Private Servers is the next best thing to a real full pledge server and more affordable. VPS is a virtual sever running on an actual server. One server ca run a few VPSs that why its more affordable. A dedicated server will cost you at least $125 per month while a VPS starts at $15. Why get a VPS? If you need to run programs that are not suitable for shared hosting and cant affordable dedicated server, VPS is a good choice.

I been lurking in forums, picking up reviews on various VPS companies. It seems that the cheaper ones have more support problem. Now it breaks down to how fast before I get my own dedicated server. If I can pull in enough income from the sites that I hosted than I will go for something cheap but if I need to use the VPS for a while than I rather get something more reliable.

Ruby on Rails, the Real Web 2.0 Programming Language

Oct 14th, 2007

Ruby on Rails is a free web application framework that aims to increase the speed and ease with which database-driven web sites can be created and offers skeleton code frameworks (scaffolding) from the outset. Ruby on Rails was created by David Heinemeier Hansson, a Danish.

Ruby on Rails is based on Ruby is a reflective, dynamic, object-oriented programming language. It combines syntax inspired by Perl with Smalltalk-like object-oriented features, and also shares some features with Python, Lisp, Dylan, and CLU. Ruby is a single-pass interpreted language. Its official implementation is free software written in C. The language was created by Yukihiro “Matz” Matsumoto, who started working on Ruby on February 24, 1993, and released it to the public in 1995. “Ruby” was named as a gemstone because of a joke within Matsumoto’s circle of friends alluding to Perl’s name

You are able to develop a web application at least ten times faster with Rails than you can with a typical Java framework. Rails takes full advantage of Ruby, which means you write fewer lines of code to implement your application. Keeping your code small means faster development and fewer bugs, which makes your code easier to understand, maintain, and enhance.

Ruby on Rails have the ability to generate ready to use codes based on your database, which is MySQL most of the time. As a complete framework, its already web 2.0 ready. Whats lacking right now is the hosting for ruby. Most hosting company do not offer Ruby on Rails. Ruby on Rails have more potential as a full web 2.0 than PHP. You can learn more about Ruby on Rails here.
