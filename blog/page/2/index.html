Grub Rescue Error

Mar 14th, 2011

I used gpart to partition a 600g hard disk with preinstall win7. When I boot up after that, boot sector for win7 was gone or maybe replaced. The grup rescue command prompt appear. I couldn’t fix it.
I simply install Linux and got a new boot menu.

When I migrate win7 to virtual box the same problem appear. This time I had to use win7 rescues disk to fix the problem. Below are the steps I found

  1. Put the Windows Vista installation disc in the disc drive, and then start the computer.
  2. Press a key when you are prompted.
  3. Select a language, a time, a currency, a keyboard or an input method, and then click Next.
  4. Click Repair your computer.
  5. Click the operating system that you want to repair, and then click Next.
  6. In the System Recovery Options dialog box, click Command Prompt.
  7. Type Bootrec.exe /FixMbr, and then press ENTER.

DVD Player Region Hack

Feb 5th, 2011

I purchase a LG dvd player sometime ago. It was a display set on sale $300 for a $500 set. Recently, it somehow cold booted, and some DVD I bought cannot be played. A message “Wrong regional code”.

I google a bit and found that I could hack the region code.

I got the instruction from videohelp.com/dvdhacks
1. Press “Home key on Remote Control

  1. Using your cursor keys, move to “Setup” and press “Enter” on the RCU

  2. Move cursor to “Lock” icon using down arrow on Remote Control

  3. Press “Zero(0)” key on Remote Control 7 times (0-0-0-0-0-0-0)

A dialogue box with a message saying “Congratulations! Region Change Fee!”

I suspect my set was set to region free by the sales guys at the store when on display.

Portable Skype

Mar 12th, 2009

I been a big fan of portable programs ever since I wipe out accidentally my firefox 2 years ago. I was looking for a way not to reinstall my applications and to make them go on my usb. I been using a couple of such program Firefox, Thunderbird, coreftp, putty, keeppass, xampp and many more. The latest addition is Skype.
Download Squad show as how

  1. Install Skype on your PC and then find the installation folder. It’s probably something like “C:Program FilesSkype.” Copy the Skype.exe file to a folder on your USB flash drive.  

  2. In that same folder on your flash drive, create a subfolder called “data.”

  3. Create a new text file called “Skype.bat” or Skype Portable.bat” or whatever you like.

  4. Open your new file using Notepad or another text editor and write “skype.exe /datapath:”data” /removable” (without the quotes).

  5. Save and close your file
