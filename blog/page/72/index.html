Free Portable Software

May 3rd, 2007 | Comments

After my Desktop died last year, I came to like free software that does not need installation. I put all my software on my USB drives and used them on any PC. I make a list of them here

  1. Coreftp - Free FTP client software for Windows
  2. CDex - for ripping cds to mp3
  3. GIMP Portable - Replacement for Adobe Photoshop
  4. Firefox Portable - my favorite browser make portable.
  5. Thunderbird Portable - A great email client
  6. Notepad++ - For editing program, a notepad with tabbed interface, configurable syntax highlighting.
  7. VLC Media Player Portable - Alternative video player
  8. Foxit PDF Reader- A PDF reader, I screw my Adobe PDF read recently and did not want to go through the trouble of uninstalling and installing
  9. ClamWin Portable - Antivirus Software.

Other Software That I should explore

  1. XAMPP is an integrated server package of Apache, mySQL, PHP and Perl.
  2. Gaim Portable - chat with AOL, MSN and Yahoo users in an easy-to-use interface

Blogging From Windows Mobile Devices

Apr 28th, 2007 | Comments

I have my 02 atom for almost a year now. Its a useful phone pda except for its ability to browse the web. The Pocket Internet Explorer are not able to view javascript or flash. I tried to post from my 02, the basic functions are there, however the publishing button does not response. The only blogging mobile client available are only for typepad. Wordpress has a interface special for mobile devices.

To blog from blogger you have purchase a browse that supports javascript. They are Netfont, Thunderhawk or opera. No client have been build mainly because google do not have the authenication api available.

The work around is to use email to blog, you have set at email id with blogger, then post to this address with text only. However if some else get hold of that email address, they can post to your blog.

If theres any blogging client that allows me to post to blogger and free, I would like to know about it.
